建立github倉儲順序

1. 利用github建立投影片

2. git clone github倉儲網址 -把倉儲內容複製下來

3. cd 資料夾名 -進入資料夾

4. git branch gh-pages -建立分支

5. git checkout gh-pages -換分支

6. 到老師的github上把reveal倉儲複製下來

7. 再把複製下來的內容丟到自己git clone 下來的資料夾(除了README.md 和.git)

8. git add-A 更改內容

9. git commit -m"40323146(學號)"

10. git config --global user.name 確認帳號

11. git config --global user.email 確認密碼

12. git push original gh-pages 推到原本網站
