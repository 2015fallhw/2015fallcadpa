SolveSpace是一個開源的2D和3D CAD（計算機輔助設計）方案。它是一個基於約束的參數化建模與簡單的機械模擬功能。




      作業
============

練習 SolveSpace

git clone 下載遠端的資料庫
clone完 在用cd 進入資料庫
利用git branch gh-pages建立分支
git checkout gh-pages切換分支
將老師的reveal.js  整包複製到分支已經切換到的倉儲中
在後再上傳
上傳指令
git add -A
git commit -m"40323116"
git push origin gh-pages
